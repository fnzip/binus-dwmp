📊 Medical Insurance Cost Data Warehouse & Dashboard

🎯 Project & Tujuan

Project ini membangun data warehouse sederhana dan dashboard interaktif berbasis Streamlit untuk menganalisis biaya asuransi kesehatan di Amerika Serikat.

  • Memberikan infografis tentang biaya rata-rata asuransi kesehatan per region.
  • Menganalisis faktor demografi (usia, jenis kelamin, jumlah anak, BMI, status merokok) terhadap biaya asuransi.
  • Menyediakan aplikasi mini tempat pengguna bisa memasukkan data (usia, sex, anak, smoker, region) untuk mendapat estimasi biaya asuransi serta insight seperti: “Jika Anda tidak merokok, biaya akan berkurang sekian USD”.
📂 Sumber Data

Dataset diambil dari Kaggle – Medical Insurance Cost Dataset
Kaggle Insurance Dataset

Dataset berisi 1338 record dengan kolom:

  • age: usia peserta
  • sex: jenis kelamin (male/female)
  • bmi: body mass index
  • children: jumlah anak yang ditanggung
  • smoker: status perokok (yes/no)
  • region: wilayah tempat tinggal (northeast, northwest, southeast, southwest)
  • charges: biaya asuransi kesehatan
🏗️ Schema yang Digunakan

Project ini menggunakan Star Schema untuk data warehouse.

  • Fact Table: fact_insurance
    Kolom: insurance_id, age_id, sex_id, bmi, children_id, smoker_id, region_id, charges
  • Dimension Tables:
    • dim_age (kategori usia: 18–25, 26–35, 36–50, 50+)
    • dim_sex (jenis kelamin)
    • dim_bmi (kategori BMI → opsional)
    • dim_children (jumlah anak)
    • dim_smoker (status perokok)
    • dim_region (wilayah tinggal)

Model ini dipilih karena sederhana, query OLAP lebih cepat, dan mudah divisualisasikan.
